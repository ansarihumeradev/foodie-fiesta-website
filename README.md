# 🍽️ Foodie Fiesta Website

A modern and responsive restaurant website developed to provide users with an engaging online food ordering experience. The project features an attractive user interface, easy navigation, and essential restaurant functionalities such as browsing the menu, learning about the restaurant, and contacting the team.

---

## 📖 Project Overview

Foodie Fiesta is a front-end web application designed to showcase a restaurant's services in a visually appealing and user-friendly manner. The website focuses on responsive design, clean layouts, and an intuitive user experience. It allows visitors to explore food items, learn about the restaurant, and get in touch through a contact page.

This project was developed as part of my academic learning to strengthen my web development skills using HTML, CSS, JavaScript, and PHP.

## 🚀 Getting Started

### Prerequisites

* Modern web browser (Chrome, Edge, Firefox, Safari)
* Visual Studio Code (recommended)

### Installation

1. Clone the repository

```bash
git clone https://github.com/ansarihumeradev/foodie-fiesta-website.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

> **Note:** If you want the PHP contact form to work, run the project using a local server such as XAMPP, WAMP, or Laragon.
